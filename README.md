# Save-Neck
由于我长时间在电脑前坐着，我的颈椎不是很好，写一个小玩意起到提示我“该休息了”的作用
## 第1版：
  只是一个基本的计时器，能够校验输入的合法性，输入计时时间，到时了会自动弹出提示窗口
## 第2版（提交信息没改过来）：
  相较于第1版功能上没有改动，第2版把计时器功能和GUI界面分开。
  还修复了两个个小问题，
    程序已经在计时的时候，再次点击开始按钮，会再启动一个计时；
    还有一个逻辑上的错误（第1版存在问题但是没有体现），第2版中在程序中止状态下，中止按钮没有功能（应该是没有功能的）
